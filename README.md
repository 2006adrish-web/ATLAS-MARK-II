# ATLAS MARK II

### AI Operating Assistant

### Powered by Google Gemini

### Developed by Adrish Industries Labs

---

## Vision

Atlas MARK II is a desktop AI operating assistant designed to bridge natural language and computer control.

Instead of manually navigating applications, files, browsers, and utilities, users interact with Atlas through voice and text while Gemini provides reasoning, planning, memory integration, and intelligent responses.

Atlas functions as an AI layer over Windows, combining automation, vision, voice interaction, memory systems, and desktop control into a unified interface.

---

## Features

### Gemini AI Integration

Atlas uses Google's Gemini models as its reasoning engine.

Capabilities include:

* General conversations
* Coding assistance
* Planning and task breakdown
* Problem solving
* Memory-aware responses
* Contextual reasoning

---

### Voice Assistant

* Speech recognition
* Natural voice responses
* Hands-free operation
* Voice command execution

---

### System Automation

Atlas can:

* Open applications
* Launch websites
* Create folders
* Create files
* Execute desktop actions
* Navigate workflows

Examples:

```text
open youtube
launch calculator
create folder AtlasProjects
create file notes.txt
```

---

### Vision System

Integrated vision modules provide:

* Screenshot capture
* Face monitoring
* Camera processing
* Future visual AI expansion

---

### Memory Engine

Atlas maintains memory using a local database.

Capabilities include:

* Storing memories
* Retrieving recent context
* Personalized interactions
* Persistent assistant state

---

### WhatsApp Integration

Atlas can send messages through connected automation modules.

Example:

```text
message John saying Hello from Atlas
```

---

### Planner Module

Generate intelligent plans for goals.

Example:

```text
plan build a SaaS startup
```

---

### Web Search

Search the internet directly from Atlas.

Example:

```text
search latest AI news
```
